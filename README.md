# Reactjs hands on tutorial: [tic tac toe](https://reactjs.org/tutorial/tutorial.html#before-we-start-the-tutorial)

### To run project:
run `npm start`
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.


### To test project:
run `npm test`

### To build project:
run `npm run build`
